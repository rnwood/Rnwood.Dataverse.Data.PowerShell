---
title: "Remove-DataverseFormTab - Remove a tab by ID with immediate publishing"
tags: ['Metadata']
source: "Remove-DataverseFormTab.md"
---
Removes the tab with the specified ID and publishes the form to make changes visible immediately.

```powershell
Remove-DataverseFormTab -FormId $formId -TabId 'tab-guid-12345' -Publish

```
