title: "Remove-DataverseSolution - Remove with confirmation"
tags: ['Solutions']
source: "Remove-DataverseSolution.md"
---
Removes the solution with explicit confirmation.

```powershell
Remove-DataverseSolution -UniqueName "MySolution" -Confirm

# Confirm
# Are you sure you want to perform this action?
# Performing the operation "Remove" on target "Solution 'MySolution'".
# [Y] Yes  [A] Yes to All  [N] No  [L] No to All  [S] Suspend  [?] Help (default is "Y"): Y
# Solution 'My Solution' removed successfully.

```
