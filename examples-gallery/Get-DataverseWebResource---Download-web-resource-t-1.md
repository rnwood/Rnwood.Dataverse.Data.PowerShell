---
title: "Get-DataverseWebResource - Download web resource to file"
tags: ['Metadata']
source: "Get-DataverseWebResource.md"
---
Downloads the web resource content to a local file. Content is automatically fetched even without -IncludeContent.

```powershell
Get-DataverseWebResource onn -Name "new_myscript" -Path "./local-script.js"

```
