---
title: "Set-DataverseEnvironmentVariableValue - Set a single environment variable value"
tags: ['Solutions']
source: "Set-DataverseEnvironmentVariableValue.md"
---
Sets the value for the environment variable "new_apiurl". The definition must already exist.

```powershell
Set-DataverseEnvironmentVariableValue -SchemaName "new_apiurl" -Value "https://api.production.example.com"

```
