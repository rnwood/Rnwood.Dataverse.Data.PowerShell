---
title: "Set-DataverseConnectionReference - Update an existing connection reference"
tags: ['Solutions']
source: "Set-DataverseConnectionReference.md"
---
Updates the connection ID of an existing connection reference. ConnectorId is not required for updates.

```powershell
Set-DataverseConnectionReference -ConnectionReferenceLogicalName "new_sharedconnectionref" -ConnectionId "87654321-4321-4321-4321-210987654321"

```
