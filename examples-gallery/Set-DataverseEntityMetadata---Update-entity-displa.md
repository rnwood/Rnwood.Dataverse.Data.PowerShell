---
title: "Set-DataverseEntityMetadata - Update entity display names"
tags: ['Metadata']
source: "Set-DataverseEntityMetadata.md"
---
Updates the display names and description of an existing entity.

```powershell
Set-DataverseEntityMetadata -EntityName new_product `
   -DisplayName "Product Catalog" `
   -DisplayCollectionName "Product Catalogs" `
   -Description "Updated product catalog description"

```
