---
title: "Set-DataverseEntityMetadata - Update icon properties on existing entity"
tags: ['Metadata']
source: "Set-DataverseEntityMetadata.md"
---
Updates only the icon properties of an existing entity without changing other properties.

```powershell
Set-DataverseEntityMetadata -EntityName account `
   -IconVectorName "svg_custom_account" `
   -IconLargeName "Custom/account_large.png"

```
