---
title: "Get-DataverseSolution - Retrieve a specific solution by unique name"
tags: ['Solutions']
source: "Get-DataverseSolution.md"
---
Retrieves a specific solution by its unique name.

```powershell
Get-DataverseSolution -UniqueName "MySolution"

# UniqueName            : MySolution
# Name                  : My Solution
# Version               : 1.0.0.0
# IsManaged             : False
# Description           : My custom solution
# PublisherName         : Contoso
# PublisherUniqueName   : contoso
# PublisherPrefix       : con

```
