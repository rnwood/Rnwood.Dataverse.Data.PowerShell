title: "Remove-DataversePluginAssembly - Remove with IfExists flag"
tags: ['Plugins']

---
Removes the plugin assembly if it exists, without raising an error if it doesn't.

```powershell
Remove-DataversePluginAssembly -Id 12345678-1234-1234-1234-123456789012 -IfExists

```
