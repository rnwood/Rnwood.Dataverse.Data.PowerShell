title: "Set-DataverseAppModule - Create a new app module"
tags: ['Metadata']
source: "Set-DataverseAppModule.md"
---
Creates a new app module with the specified unique name, display name, and description.

```powershell
Set-DataverseAppModule -PassThru `
   -UniqueName "myapp_unique" `
   -Name "My Custom App" `
   -Description "A custom model-driven application"

```
