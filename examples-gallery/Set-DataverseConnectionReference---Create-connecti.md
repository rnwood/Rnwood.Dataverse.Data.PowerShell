---
title: "Set-DataverseConnectionReference - Create connection reference with description"
tags: ['Solutions']
source: "Set-DataverseConnectionReference.md"
---
Creates a connection reference with display name and description.

```powershell
Set-DataverseConnectionReference -ConnectionReferenceLogicalName "new_sqlconnection" -ConnectionId "12345678-1234-1234-1234-123456789012" -ConnectorId "98765432-4321-4321-4321-210987654321" -DisplayName "Production SQL Database" -Description "Connection to the production SQL database server"

```
