---
title: "Set-DataversePluginAssembly - Create a new plugin assembly from file"
tags: ['Plugins']
source: "Set-DataversePluginAssembly.md"
---
Creates a new plugin assembly by reading from a file with external isolation mode.

```powershell
Set-DataversePluginAssembly -Name "MyPlugin" -FilePath "C:\Plugins\MyPlugin.dll" -IsolationMode 2

```
