---
title: "Set-DataverseConnectionReference - Create a new connection reference"
tags: ['Solutions']
source: "Set-DataverseConnectionReference.md"
---
Creates a new connection reference with the specified logical name, connection ID, connector ID, and display name.

```powershell
Set-DataverseConnectionReference -ConnectionReferenceLogicalName "new_sharedconnectionref" -ConnectionId "12345678-1234-1234-1234-123456789012" -ConnectorId "98765432-4321-4321-4321-210987654321" -DisplayName "Shared SharePoint Connection"

```
