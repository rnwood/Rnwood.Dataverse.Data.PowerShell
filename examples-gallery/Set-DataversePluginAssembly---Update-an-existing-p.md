---
title: "Set-DataversePluginAssembly - Update an existing plugin assembly"
tags: ['Plugins']
source: "Set-DataversePluginAssembly.md"
---
Updates an existing plugin assembly with new content and version.

```powershell
Set-DataversePluginAssembly -Id 12345678-1234-1234-1234-123456789012 -Name "MyPlugin" -FilePath "C:\Plugins\MyPlugin.dll" -Version "2.0.0"

```
