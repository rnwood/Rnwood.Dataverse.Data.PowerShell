# Remove-DataverseMsAppDataSource

## SYNOPSIS
Removes a data source from a Canvas app .msapp file.

## SYNTAX

### FromPath
```
Remove-DataverseMsAppDataSource [-MsAppPath] <String> [-DataSourceName] <String> [-IfExists]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### FromObject
```
Remove-DataverseMsAppDataSource [-CanvasApp] <PSObject> [-DataSourceName] <String> [-IfExists]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes a named data source from a Canvas app .msapp file.

> **EXPERIMENTAL:** YAML-first Canvas app modification is experimental. The Power Apps MSAPP format it not fully understood/supported and the results may need to be validated in Power Apps Studio.

## EXAMPLES

### Example 1: Remove a data source
```powershell
Remove-DataverseMsAppDataSource -MsAppPath "myapp.msapp" -DataSourceName "Accounts"
```

### Example 2: Remove if exists (no error if not found)
```powershell
Remove-DataverseMsAppDataSource -MsAppPath "myapp.msapp" -DataSourceName "OldDataSource" -IfExists
```
