# Set-DataverseBot

## SYNOPSIS
Creates or updates a Copilot Studio bot in Dataverse.

## SYNTAX

```
Set-DataverseBot [-BotId <Guid>] -Name <String> [-SchemaName <String>] [-Language <Int32>]
 [-Configuration <String>] [-AuthenticationMode <Int32>] [-RuntimeProvider <Int32>] [-Template <String>]
 [-PassThru] [-Connection <ServiceClient>] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Creates a new Copilot Studio bot or updates an existing one in Dataverse. If BotId is provided, updates the existing bot. If BotId is not provided, creates a new bot with the specified properties.

## EXAMPLES

### Example 1
```powershell
PS C:\> $bot = Set-DataverseBot -Name "Support Bot" -SchemaName "support_bot" -Language 1033 -PassThru
```

Creates a new bot with English language (1033) and returns the created bot object.
