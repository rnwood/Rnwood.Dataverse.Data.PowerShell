# Get-DataverseMsAppControlTemplateProperty

## SYNOPSIS
Lists properties for a Canvas control template from embedded all-controls metadata, optionally augmented by a .msapp file.

## SYNTAX

### Embedded (Default)
```
Get-DataverseMsAppControlTemplateProperty [-TemplateName] <String> [-TemplateVersion <String>]
 [-IncludeContextualDynamicProperties] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### FromPath
```
Get-DataverseMsAppControlTemplateProperty [-TemplateName] <String> [-TemplateVersion <String>]
 [-MsAppPath] <String> [-IncludeContextualDynamicProperties] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

### FromObject
```
Get-DataverseMsAppControlTemplateProperty [-TemplateName] <String> [-TemplateVersion <String>]
 [-CanvasApp] <PSObject> [-IncludeContextualDynamicProperties] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION
Returns property metadata for a specific control template.
By default, template metadata is read from the embedded `All controls.msapp` catalog.
If `-MsAppPath` or `-CanvasApp` is provided, those templates are merged with embedded templates.

If `-TemplateVersion` is not provided, the latest version across all selected sources is used.

The output includes one row per property with template metadata and default value.
Use `-IncludeContextualDynamicProperties` to include auto-layout child properties injected by runtime.

## EXAMPLES

### Example 1: List properties for the latest button template
```powershell
Get-DataverseMsAppControlTemplateProperty -TemplateName "button"
```

### Example 2: Inspect a specific template version
```powershell
Get-DataverseMsAppControlTemplateProperty -TemplateName "label" -TemplateVersion "2.5.1"
```

### Example 3: Include contextual auto-layout child properties
```powershell
Get-DataverseMsAppControlTemplateProperty -TemplateName "button" -IncludeContextualDynamicProperties
```

### Example 4: Use templates from a local .msapp file as an additional source
```powershell
Get-DataverseMsAppControlTemplateProperty -TemplateName "button" -MsAppPath "C:\apps\myapp.msapp"
```
