# Export-DataverseBot

## SYNOPSIS
Exports a complete bot backup to a directory.

## SYNTAX

```
Export-DataverseBot [-BotId] <Guid> [[-OutputPath] <String>] [-PassThru] [-Connection <ServiceClient>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Exports a Copilot Studio bot configuration and all its components to a structured directory. Creates YAML data files and JSON metadata files compatible with PiStudio-CLI backup format.

## EXAMPLES

### Example 1
```powershell
PS C:\> $export = Export-DataverseBot -BotId $bot.botid -PassThru
```

Exports the bot to an auto-generated timestamped directory and returns export info.
