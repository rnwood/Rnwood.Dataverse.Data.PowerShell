# Remove-DataverseBot

## SYNOPSIS
Deletes a Copilot Studio bot from Dataverse.

## SYNTAX

```
Remove-DataverseBot [-BotId] <Guid> [-Connection <ServiceClient>] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Permanently deletes a Copilot Studio bot record from Dataverse. This operation requires confirmation by default due to its high impact.

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-DataverseBot -BotId $bot.botid
```

Deletes the specified bot after confirmation prompt.
