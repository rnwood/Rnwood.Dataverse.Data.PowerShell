# Invoke-DataverseImmediateBook

## SYNOPSIS
Contains the data to book an appointment transactionally, obeying the constraints specified by the associated service and the supplied appointment request.

[Microsoft Learn: Microsoft.Crm.Sdk.Messages.ImmediateBookRequest](https://learn.microsoft.com/dotnet/api/Microsoft.Crm.Sdk.Messages.ImmediateBookRequest)

## SYNTAX

```
Invoke-DataverseImmediateBook -Connection <ServiceClient> -Target <PSObject> -TargetTableName <String> -TargetIgnoreProperties <String[]> -TargetLookupColumns <Hashtable> -AttachmentIds <String[]> -EWSUrl <String> -AttachmentToken <String>
```

## DESCRIPTION
Contains the data to book an appointment transactionally, obeying the constraints specified by the associated service and the supplied appointment request.

## EXAMPLES

### Example 1
```powershell
PS C:\> Invoke-DataverseImmediateBook -Connection <ServiceClient> -Target <PSObject> -TargetTableName <String> -TargetIgnoreProperties <String[]> -TargetLookupColumns <Hashtable> -AttachmentIds <String[]> -EWSUrl <String> -AttachmentToken <String>
```
