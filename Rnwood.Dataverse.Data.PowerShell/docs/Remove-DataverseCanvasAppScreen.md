# Remove-DataverseCanvasAppScreen

## SYNOPSIS
Removes a screen from a Canvas app's .msapp file.

## SYNTAX

```
Remove-DataverseCanvasAppScreen -CanvasAppId <Guid> [-ScreenName] <String> [-IfExists]
 [-Connection <ServiceClient>] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes a screen from a Canvas app's .msapp file by modifying the app's document content.

## EXAMPLES

### Example 1: Remove a screen from a Canvas app
```powershell
PS C:\> Remove-DataverseCanvasAppScreen -CanvasAppId "12345678-1234-1234-1234-123456789012" -ScreenName "OldScreen"
```

Removes a screen called "OldScreen" from a Canvas app.
