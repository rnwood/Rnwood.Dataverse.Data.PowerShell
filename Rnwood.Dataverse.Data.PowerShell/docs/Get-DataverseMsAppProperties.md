# Get-DataverseMsAppProperties

## SYNOPSIS
Gets app-level properties from a Canvas app .msapp file.

## SYNTAX

### FromPath
```
Get-DataverseMsAppProperties [-MsAppPath] <String> [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### FromObject
```
Get-DataverseMsAppProperties [-CanvasApp] <PSObject> [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Reads `Src/App.pa.yaml` from a Canvas app .msapp file and returns the app properties content.
Supports reading directly from a file path or from a Canvas app object returned by
`Get-DataverseCanvasApp -IncludeDocument`.

## EXAMPLES

### Example 1: Get app properties from a local .msapp file
```powershell
PS C:\> Get-DataverseMsAppProperties -MsAppPath "myapp.msapp"
```

Returns app-level YAML properties from `Src/App.pa.yaml`.

### Example 2: Get app properties from pipeline
```powershell
PS C:\> $app = Get-DataverseCanvasApp -Name "myapp" -IncludeDocument
PS C:\> $app | Get-DataverseMsAppProperties
```

Reads the .msapp document bytes from the Canvas app object and returns app-level properties.
