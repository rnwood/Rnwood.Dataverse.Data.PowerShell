# Invoke-DataverseWinQuote

## SYNOPSIS
Contains the data that is needed to set the state of a quote to Won.

[Microsoft Learn: Microsoft.Crm.Sdk.Messages.WinQuoteRequest](https://learn.microsoft.com/dotnet/api/Microsoft.Crm.Sdk.Messages.WinQuoteRequest)

## SYNTAX

```
Invoke-DataverseWinQuote [-QuoteClose <PSObject>] [-QuoteCloseTableName <String>]
 [-QuoteCloseIgnoreProperties <String[]>] [-QuoteCloseLookupColumns <Hashtable>] [-Status <OptionSetValue>]
 [-Connection <ServiceClient>] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Contains the data that is needed to set the state of a quote to Won.

## EXAMPLES

### Example 1
```powershell
PS C:\> Invoke-DataverseWinQuote -Connection <ServiceClient> -QuoteClose <PSObject> -QuoteCloseTableName <String> -QuoteCloseIgnoreProperties <String[]> -QuoteCloseLookupColumns <Hashtable> -Status <OptionSetValue>
```
